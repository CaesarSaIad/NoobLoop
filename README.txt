External Modules required: 
1 - Pygame: Install from http://www.pygame.org/hifi.html
2 - Numpy: Install from http://www.numpy.org/
3 - Scipy.io.wavfile: Install from http://scipy.org/scipylib/index.html

-All other modules should either be already built into Python 
(e.g. tkinter) or included in termProject.zip (e.g. snd110, dialogboxdemo).
-Sources and usages of all external modules cited in noobLoop3.py

Welcome to NoobLoop: the audio looper made simple.

Run noobLoop3.py to begin. 

Load "tester" to preview a close-to-complete project.
